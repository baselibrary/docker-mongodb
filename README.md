## ThoughtWorks Docker Image: mongodb

[![](http://dockeri.co/image/baselibrary/mongodb)](https://registry.hub.docker.com/u/baselibrary/mongodb/)

### Base Docker Image

* `latest`: mongodb 3.0
* `3.0`   : mongodb 3.0
* `2.6`   : mongodb 2.6

### Installation

    docker pull baselibrary/mongodb

### Usage

    docker run -it --rm baselibrary/mongodb
